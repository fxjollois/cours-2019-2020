Cartographie sous python¶

Dans ce TP, nous allons aborder l'aspect cartographie sous python avec le package folium ( installer donc). L'ide sera d'ajouter un fond de carte des formes de couleurs, en fonction d'une information tierce.

Création de cartes¶

Le package folium est une interface entre python et la librairie leaflet. Elle s'utilise trs facilement, comme nous pouvons le voir ci-dessous pour la cration d'une carte. Les coordonnes indiques sont la latitude et la longitude de New-York.

In [1]:
import folium
centre = [40.71427, -74.00597]
newyork = folium.Map(location = centre, zoom_start = 12)

Pour obtenir la carte, il suffit d'appeler l'objet ainsi cr.

Ajout de marqueurs¶

La fonction Marker() permet d'ajouter une marque sur la carte pour signifier un emplacement prcis. Il est possible de spcifier un contenu afficher (sous la forme d'un pop-up) lorsque l'utilisateur va cliquer sur le marqueur. Celui-ci sera plac la localisation fournie en premier, au format [latitude, longitude]. Ce contenu est spcifi dans l'argument popup (ici, une simple chane de caractres, mais cela peut tre plus labor). Il est aussi possible customiser l'icne du marqueur. Une fois ce marqueur cr, il faut l'ajouter la carte, via la fonction add_to().

In [3]:
folium.Marker(centre, popup = "New-York").add_to(newyork)
newyork
Out[3]:

Ajout de cercles¶

La fonction CircleMarker() permet elle de crer des cercles, en donnant la localisation du centre et le rayon (via le paramtre radius). Idem que pour Marker(), on peut spcifier le contenu de la popup affiche lors d'un clic, et il est ncessaire de l'ajouter la carte. Il est possible aussi de spcifier la couleur du cercle (via color) et la couleur de remplissage (via fillColor).

In [4]:
folium.CircleMarker(centre, popup = "New-York", radius = 10).add_to(newyork)

Avec une information tierce¶

Bien videmment, nous dsirons ajouter une couleur. Pour cela, nous avons besoin d'un DataFrame contenant deux variables :

  • un identifiant commun avec les informations contenues dans le GeoJSON
  • la variable statistique d'intrt (pour cet exemple, ce sera la surface)

Pour le construire, nous allons rcuprer le nom du quartier (item BoroName dans properties) et la surface (Shape_Area dans properties). Voici comment les rcuprer.

Pour relier ces informations la carte, toujours dans la fonction choropleth(), nous devons dfinir les lments suivants :

  • geo_data : les contours au format GeoJSON
  • key_on : l'item dans le GeoJSON avec lequel nous ferons la jointure
  • data : le DataFrame dans lequel nous avons les informations statistiques
  • columns : les deux colonnes prendre
    1. cl de jointure
    2. mesure statistique
  • fill_color : la palette de couleurs utiliser (provenant de Color Brewer)

Nous obtenons maintenant la carte suivante.

In [21]:
newyork = folium.Map(location = centre, zoom_start = 10)
newyork.choropleth(geo_data = geo, key_on = "feature.properties.BoroName",
                   data = df, columns = ["Quartier", "Surface"],
                   fill_color = 'Spectral',
                   legend_name = "Surface du quartier de New-York")
