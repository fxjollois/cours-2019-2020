Description des variables¶

Dans ce cadre, il faut dcrire chaque variable sparement. Voici la rpartition des variables et les tapes raliser pour chaque type de variable :

  • quantitative : carat, depth, table, price, x, y, z
    • Moyenne, cart-type, mdiane, minimum et maximum
    • Histogramme pour avoir un aperu de la distribution, et ventuellement comparer une gaussienne
    • Test de normalit ajouter
    • Bote moustaches pour estimer la prsence ou non d'outliers
  • qualitative : cut, color, clarity
    • Table d'effectifs et de pourcentages
    • Diagramme en barres (diagramme circulaire viter ici, car trop de modalits)

Nous allons faire une fonction permettant la description d'une variable (en fonction de son type bien videmment), et l'utiliser pour chacune des colonnes du jeu de donnes.

def descQuanti(var):
    # description d'une variable quantitative du jeu de donnes diamonds
    # rcupration des valeurs
    x = diamonds[var]
    # affichage des diffrentes informations demandes
    print("Variable   :", var, "\n")
    print("Moyenne    : ", numpy.mean(x).round(2))
    print("Ecart-type : ", numpy.std(x).round(2))
    print("Mdiane    : ", numpy.median(x).round(2))
    print("Minimum    : ", numpy.min(x).round(2))
    print("Maximum    : ", numpy.max(x).round(2))
    # cration d'un graphique avec 2 zones
    fig, (ax1, ax2) = matplotlib.pyplot.subplots(nrows=1, ncols=2, figsize=(8, 4))
    # cration de l'histogramme avec la densit
    h = x.plot(kind = "hist", density = True, color = "lightgrey", ax = ax1)
    x.plot(kind = "kde", ax = ax1)
    # on y ajoute la p-value du test de normalit
    t = scipy.stats.normaltest(x)
    matplotlib.pyplot.text(.9, .9, "Normalit (p-val) : " + str(t.pvalue),
                           horizontalalignment='right',
                           verticalalignment='top',
                           transform = h.transAxes)
    # cration de la bote  moustache
    diamonds.boxplot(column = var, grid = False, ax = ax2)

def descQuali(var):
    # description d'une variable qualitative du jeu de donnes diamonds
    # rcupration des valeurs
    x = diamonds[var]
    # affichage des diffrentes informations demandes
    print("Variable   :", var, "\n")
    # tableau des effectifs et des proportions
    eff = pandas.crosstab(x, "Effectif")
    pro = pandas.crosstab(x, "Pourcentage", normalize=True) * 100
    print(pandas.concat([eff, pro.round(2)], axis = 1))
    # diagramme en barres
    eff.plot(kind = "bar", legend = False)

# Cration d'une fonction pour tablir le lien entre price et une variable quali
def lienPriceQuali(var):
    # on rcupre les valeurs de la variable
    x = diamonds[var]
    # cration des botes  moustaches
    b = diamonds.boxplot(column = "price", by = var, grid = False) 
    # ajout d'un espace en haut du graphique pour y mettre la p-value de l'ANOVA
    b.set_ylim(0, numpy.max(diamonds.price) * 1.25)
    # calcul de l'ANOVA
    g = [diamonds.price[x == s] for s in list(x.unique())]
    aov = scipy.stats.f_oneway(*g)
    # ajout de la p-value dans le graphique
    matplotlib.pyplot.text(.5, .9, "ANOVA (p-value) : " + str(aov.pvalue.round(5)),
                           horizontalalignment='center',
                           verticalalignment='top',
                           transform = b.transAxes)
