Statistiques sous Python¶

Nous allons utiliser dans ce TP le module pandas permettant l'analyse de donnes avec Python. La premire instruction est d'installer le module, faire dans un terminale de commande. Nous devons aussi installer les modules matplotlib et scipy.

{bash}
pip3 install pandas
pip3 install matplotlib
pip3 install scipy
pip3 install numpy

Une fois ces modules installs, nous pouvons lancer un notebook pour commencer notre programme.

Il faut tout d'abord importer ces modules. La dernire ligne permettra de voir le rsultat des graphiques dans le document.

In [1]:
import matplotlib.pyplot
import pandas
import scipy.stats
import numpy

%matplotlib inline

Histogramme¶

Pour reprsenter graphiquement cette variable, pandas met disposition (via le module matplotlib utilis par pandas) des fonctions graphiques.

Pour raliser un histogramme, nous utilisons la fonction hist(). Celle-ci peut prendre des options. La fonction plot() avec le paramtre kind avec la valeur "hist" revient au mme rsultat.

tips.total_bill.plot(kind = "kde")
Out[28]:
